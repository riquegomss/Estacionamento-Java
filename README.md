# 🚗 Sistema de Gerenciamento de Estacionamento

Este é um sistema de gerenciamento de estacionamento desenvolvido em **Java**, focado na simplicidade e eficiência para controle de fluxo de veículos. O projeto foi estruturado utilizando o ambiente de desenvolvimento Eclipse e conta com persistência de dados em arquivo local.

---

## 🛠️ Tecnologias e Ferramentas

* **Linguagem:** Java
* **IDE de Desenvolvimento:** Eclipse
* **Persistência:** Armazenamento de dados local (`EST.DAT`)

---

## 📂 Estrutura do Projeto

O repositório está organizado da seguinte forma:

* `src/Main.java`: Ponto de entrada (classe executável) do sistema.
* `src/Estacionamento.java`: Contém a lógica de negócios, regras de entrada, saída e tarifas do estacionamento.
* `EST.DAT`: Arquivo binário/dados para a persistência e salvamento do estado do estacionamento.
* `grupo.txt`: Arquivo auxiliar de configuração ou metadados do projeto.
